E-Commerce Admin Dashboard 

This project features a clean, responsive interface, interactive charts, and a complete set of management tools for products, orders, and customers.

A modern, high-performance admin dashboard for managing an e-commerce platform. Built with React 19, Vite, and Tailwind CSS.
